- **Step 1 est present :**
    - *OBLIGATOIRE :* 
        - dataset
    - *FACULTATIVE:*

- **Step 2 est present :**
    - *OBLIGATOIRE :*

    - *FACULTATIVE :*
        - loss

- **Step 3 est present :** 
    - *OBLIGATOIRE :* 
        - Step 2 | saved_model.pkl present dans le repertoire
        - Step 1 | Step 2 | dataToPredict


**params :** 
- --steps -s [int]
- --dataset -ds str
- --valPart -v float
- --layer -la [int et str]
- --loss -lo str
- --learningRate -lr float
- --batchSize -bs int 
- --epochs -e int 
- --dataToPredict -dtp str
